# New-RandomFilesAndFolders

New-RandomFilesAndFolders is a Powershell function to create a random folderstructure with a given depth.

## Example

### Code:
```
New-RandomFilesAndFolders -Path C:\Temp\dummy\ -Depth 5 -MinFolderCount 3 -MaxFolderCount 5
```
